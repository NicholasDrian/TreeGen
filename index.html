TreeGen Documentation

Parametric Tree Generation for Rhino Grasshopper

by Nick Drian

TreeGen is a collection of tools for generating highly customizable virtual trees within Rhino. The software is in a beta testing phase and will be updated frequently. Stay tuned for version 1.0 which is to be released in May of 2022!

There are currently two main tree generators; one that specializes in Pine Trees, and another for deciduous trees. Descriptions and downloads of these two tree generators can be found below, along with a gallery of examples.
